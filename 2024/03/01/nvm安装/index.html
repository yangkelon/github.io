nvm安装

①、如之前电脑中安装了node，先把node删除干净。

②、下载nvm安装包，解压后点击exe文件安装，点击下一步安装到D:\NVM下，点击下一步在创建nodejs文件夹D:\NVM\nodejs，点击完成安装。

④、配置环境变量

​ 用户变量：

​ NVM_HOME D:\NVM

​ NVM_SYMLINK D:\NVM\nodejs

​ 系统变量：

​ NVM_HOME D:\NVM

​ NVM_SYMLINK D:\NVM\nodejs

⑤、nvm使用和常见命令

nvm list available 查看所有可以安装的node.js版本^ 1

nvm install 版本号 安装对应版本的node

nvm list 查看已安装的node

nvm use 版本号，切换到指定版本
