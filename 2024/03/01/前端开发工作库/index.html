前端开发工作库

Lodash：通过降低对array、number、objects、string ……的使用难度，让 JavaScript 变得更简单，大幅度提高前端 JavaScript 编程的效率和可读性，使得代码编写更加快捷。

xijs：丰富的工具函数，让开发更高效。

Ramda：基于柯里化的工具库，可以用于处理集合、数组、对象等数据。

qs.js：将url 参数转化 (parse 和 stringify)的轻量级 js 库。

HLS.js：基于 JavaScript 的 HTTP Live Streaming（HLS）播放器库，可用于在浏览器中播放 HLS 视频流。

Shepherd：为应用创建新手引导。

driver.js：初次进入系统的演示提示。

Bowser：获取浏览器信息。

big.js：处理大数、小数的运算操作。

Decimal.js：是一个支持JavaScript浮点运算的高精度类型，可以避免舍入规则带来的精度问题。

Validator.js：JavaScript 表单验证库。

Day.js：解析、验证、操作和显示日期和时间。

moment：解析、验证、操作和显示日期和时间。

Fecha：日期处理。

animate：动画功能。

dynamics.js：物理动画。

screenfull.js：全屏。

cropper.js：对元素进行任意角度的旋转动画。

rotate.js：图片裁剪工具。

ckeditor.js：富文本编辑。

jquery.minicolors：网页颜色选取特效。

jquery.slimscroll.js：滚动条插件。

pace.js：进度条。

webuploader.js：文件上传。

diff2html：将 Git diff 或 SVN diff 转换为漂亮的 HTML 格式的 JavaScript 库。使用 diff2html，你可以将 Git diff 或 SVN diff 的文本内容转换为 HTML 格式的展示方式，以便更好地展示和分享你的代码变更。

history：路由操作。
