Docker基础概念

①Image（镜像）：用来创建容器，Image和Container相当于类和实例。Image 包含运行Container 的配置文件，Image通过 Container 运行。Image可以通过拉取Docker Hub或其他镜像仓库现成的镜像，或通过Dockerfile手动创建。

②Container（容器）：Container可以被启动、开始、停止、删除，每个容器都是独立的运行环境。

③Compose：Docker Compose是一个定义和运行多容器工具，通过yml文件配置应用服务，只需要使用一条命令就能启动、停止所有的服务。

Docker安装MySQL

打开docker hub，查看版本，安装需要的版本，也可以用docker search mysql查看可用版本。

①docker pull mysql：拉取mysql镜像

②docker images：查看已安装的mysql

③运行容器：docker run –name mysql-test -p 3306:3306 -e MYSQL_ROOT_PASSWORD=123456 -d mysql:xxx

Docker安装Nginx（Web项目的部署容器）

①docker pull nginx：拉取nginx镜像

②docker run –name nginx-container -p 8080:80 -d nginx

③打开浏览器访问8080端口，访问nginx服务。
