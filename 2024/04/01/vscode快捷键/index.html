vscode快捷键

1、感叹号 ‘ ! ‘

输入“ ! ”，按 Enter 将获得 HTML 模板代码。

2、ID 和 CLASS 属性

输入“ # ”+id值或“ . ”+class值。将生成带有 id 、 class 的

元素。如“.navbar#header” 将生成一个 class=navbar 和 id=header 的
元素。

3、兄弟: + 和文本: { }

如“div+p+a”，将生成

4、子元素: >

如”ul>li“，将生成

5、乘号: *

如”ul>li*5“，将生成5个li
