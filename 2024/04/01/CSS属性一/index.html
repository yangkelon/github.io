CSS属性一

发表于2024-04-01更新于2024-07-07

CSS属性一

word-spacing：字间距，字之间有空格才会生效

letter-spacing：字母间距

text-transform：字母大小写

text-indent：文本首行缩进

white-space：处理元素内的空白

list-style：list-style-type、list-style-image、list-style-position：列表项标记样式、列表项标记位置

border-collapse：表格的边框是否被合并为一个单一的边框

border-spacing：表格边框边距

长度单位
  • 相对单位：px(同一设备像素相对固定)、em(根据父元素的字体大小计算)、rem(根据根元素的字体大小计算)、vw(视口的宽度)、vh(视口的高度)、vmin、vmax、ex、ch
  • 绝对单位：pt、cm、mm、in、pc
伪元素
伪元素功能描述
::after::after 会创建一个伪元素，作为所选元素的最后一个子元素。通过 content属性来为一个元素添加修饰性的内容。此元素默认是行级的。
::backdrop**::backdrop**全屏模式下，修改下层元素。
::before::before 创建一个伪元素，其将成为匹配选中的元素的第一个子元素。常通过 content属性来为一个元素添加修饰性的内容。此元素默认是行级的。
::first-letter**::first-letter**会选中某块级元素第一行的第一个字母，并且文字所处的行之前没有其他内容（如图片和内联的表格）。
::first-line**::first-line**会选中某块级元素第一行。第一行的长度取决于很多因素，包括元素宽度，文档宽度和文本的文字大小。
::marker**::marker**匹配列表的标记框。它作用在任何设置了 display: list-item的元素或伪元素上。
::placeholder::placeholder 表示input或textarea元素中的占位文本。
::selection::selection CSS 伪元素应用于文档中被用户高亮的部分（比如使用鼠标或其他选择设备选中的部分）。
伪类
伪类功能描述
:active:active 匹配被用户激活的元素。当用鼠标交互时，它代表的是用户按下按键和松开按键之间的时间。
:checked**:checked**表示任何处于选中状态的radio、 checkbox、select元素中的option。
**:disabled ****:disabled **表示任何被禁用的元素。
enabledenabled 表示任何已启用的元素。
:first-child**:first-child**表示在一组兄弟元素中的第一个元素。
:first-of-type:first-of-type 表示一组兄弟元素中其类型的第一个元素。
:fullscreen**:fullscreen**匹配当前处于全屏模式的所有元素。
:focus**:focus**表示获得焦点的元素（如表单输入）。
:hover:hover 鼠标指针悬停在元素上时触发。
:has()**:has()**伪类的是父级元素是否包含特定的子元素。如果包含则生效该样式效果，否则不生效。
:is()**:is()**将选择器列表作为参数，并选择该列表中任意一个选择器可以选择的元素。
:last-child:last-child 代表一组兄弟元素中的最后元素。
:last-of-type:last-of-type 表示一组兄弟元素中其类型的最后一个元素。
:link:link 表示尚未被访问的元素。
:not()**:not()**用来匹配不符合一组选择器的元素。
:nth-child():nth-child() 根据元素在父元素的子元素列表中的索引来选择元素。
:nth-last-child():nth-last-child() 从兄弟节点中从后往前匹配处于某些位置的元素。
:target:target 表示一个唯一的元素（目标元素），其id与当前 URL 片段匹配。
:visited:visited 用户访问链接后生效，出于隐私保护的原因，使用该选择器可以修改的样式非常有限。仅适用于带有href属性的a、area元素。
:where()**:where()**将选择器列表作为参数，并选择该列表中任意一个选择器可以选择的元素。
