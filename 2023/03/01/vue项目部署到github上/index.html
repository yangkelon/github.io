vue项目部署到github上

先打包vue项目，在打包之前修改一下vue.config.js中的publicPath属性值为’./‘，如果没有这个属性名，自己添加一下[^ 1]。

github新建一个repository，

git init
git add README.md
git commit -m “first commit”
git branch -M main
git remote add origin git@github.com:yangkelon/ysj-blog.git
git push -u origin main

打开github上该项目的Setting，点击侧边栏的Pages，选择Branch，切换到该项目的分支，选择文件夹，保存即可。

[^ 1]: vue-cli 2.0版本修改config/index.js中的assetsPublicPath；3.0版本修改vue.config.js中的baseUrl；4.0、5.0修改vue.config.js中publicPath。
